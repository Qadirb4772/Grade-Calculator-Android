# Grading Calculator Android 📱

A lightweight and efficient Android application designed for university students to calculate their academic grades and GPA/CGPA with ease.

---

## 🚀 Features
* **GPA Calculation:** Calculate Semester GPA based on credit hours and marks.
* **CGPA Tracker:** Keep track of overall academic performance.
* **Grade breakdown:** Automatically converts marks into grades (A, B, C, etc.) based on standard university scales.
* **User-Friendly UI:** Simple and clean interface built for fast input.

## 🛠️ Tech Stack
* **Language:** Java ☕
* **IDE:** Android Studio 🛠️
* **UI Design:** XML (Material Design)
* **Build Tool:** Gradle

---

## 📂 Project Structure
This project follows the standard Android directory structure:
* `app/src/main/java/`: Contains all Java logic files.
* `app/src/main/res/`: Contains layouts, strings, and drawable resources.
* `.gitignore`: Configured to exclude heavy Gradle binaries and local IDE settings.
